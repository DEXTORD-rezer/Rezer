HWID Lock Manager
HWID (Hardware ID) dikunci otomatis saat key pertama kali digunakan di sebuah PC. Key yang sama tidak bisa dipakai di PC lain. Anda bisa reset HWID dari sini.

API Documentation
Base URL: https://api.keyshield.yourdomain.com/v1
Semua request menggunakan Content-Type: application/json
POST/auth/verify
Verifikasi license key & lock HWID secara otomatis
{ "key": "CHIK-XXXX-XXXX-XXXX", "hwid": "A1B2C3D4E5F6..." } // Response sukses: { "valid": true, "hwid_locked": true, "expires": "2025-12-31", "message": "Key valid" } // Response gagal: { "valid": false, "message": "HWID mismatch - key sudah dipakai di PC lain" }
GET/key/info?key=CHIK-XXXX
Cek informasi key tanpa melakukan auth
{ "key": "CHIK-XXXX-XXXX-XXXX", "status": "active", "expires": "2025-12-31", "hwid": "locked", "duration": "30d" }
POST/key/reset-hwid
Reset HWID lock (butuh admin token)
{ "admin_token": "YOUR_ADMIN_TOKEN", "key": "CHIK-XXXX-XXXX-XXXX" } // Response: { "success": true, "message": "HWID berhasil direset" }
Contoh Implementasi di Script Anda
-- Lua / Roblox / Game Script Example local function checkLicense(key, hwid) local res = HttpPost("https://api.keyshield.yourdomain.com/v1/auth/verify", { key = key, hwid = hwid -- ambil dari GetHardwareId() atau fingerprint }) local data = json.decode(res) if data.valid then print("License valid! Expires: " .. data.expires) return true else print("License gagal: " .. data.message) return false end end

LOGIC HWID Lock — Alur Validasi
Server otomatis register HWID saat pertama kali, lalu lock ke PC tersebut.
// STEP 1 → User input key → C++ kirim {key + hwid} ke server // STEP 2 → Server cari key di database ✗ Key tidak ada → {"success":false, "message":"Invalid Key"} ✗ Key expired → {"success":false, "message":"Key Expired"} ✗ Key banned → {"success":false, "message":"Key Banned"} // STEP 3 → Cek HWID di database ✓ HWID = null/kosong → REGISTER hwid sekarang → {"success":true} ✓ HWID = cocok → {"success":true, "message":"Access Granted"} ✗ HWID ≠ beda PC → {"success":false, "message":"HWID Mismatch"}
